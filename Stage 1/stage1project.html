The Web and HTML

Round 1: Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language; it's the language of the internet. Although the internet can contain a vast array of file types, HTTP is the most dominant.

How Does the Internet Work?

Beginning on a brower a user submits a request to the internet which gets sent through HTTP to a server. The server relays back the requested information to the internet which passes it back to the user.

Common Elements and Tags

HTML uses tags to organize content. Tags contain elements which are either inline or block. Block elements are contained by an invisible box which can displace text, whereas inline elements stay inline with the text.

A sample list of tags introduced in this lesson are:

  • <img src = " " alt="text">
  • <b>bold</b>
  • <em>emphasize</em>
  • <a href='www.google.com'>link</a>
Whitespace

While coding in HTML whitespace is condensed down to a single space. Something like "hello     world" would condense to "hello world".

Practice Makes Perfect

It is not required to effortlessly recall everything taught through this course. There are resources available to brush up on forgotten HTML. There are also areas to practice online.

Round 2: I'm Stuck in a Box!

Boxes, Boxes.. So Many Boxes

Desinging an HTML website involves the use of many "invisible boxes". These boxes seperate the content and help organize the flow of the site (think of how you would organize information on a poster board).

To CSS or Not to CSS

CSS stands for Cascading Style Sheets. Similar to the decorating a house, CSS decorates websites. Always, always, use CSS when coding. Without CSS webpages look akin to their ancient, lackluster 1995 origins.

Just Your Friendly Neighborhood Text Editor

Text Editors allow for a much easier time when writing and debugging code. It's not just for HTML and CSS, this is for practically all languages!

Round 3: I'm Gonna CSS Like It's (Not) 1999!

Rules

CSS stylizes different aspects of HTML with the most specific elements gaining priority.

Where Shall We CSS?

CSS can either be written into the html document or be <link>(ed) to a separate CSS file. If it's part of the html coding it can either be placed in the <head> or inline with <style>.

Boxception

Each box has a series of smaller boxes until it reaches the content. Starting from outside to in: Margin, Border, Padding, Cotent

The benefits of working with HTML is changes to code can be seen immediately. A lot of times there will be a set of specifications that have to be met exactly. Keep testing out different things to test out what works and what doesn't. Match the requested design as best as possible.
