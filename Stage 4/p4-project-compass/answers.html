Reverse Engineering Project

Directions: Identify which type of programmer(Front End, Back end, Mobile Programmer or Data Analyst) would work on the features of Project Compass. Type your answers between the paragraph tags and be sure to provide an brief explanation for your choice.

1. Which programmer would build and manage the database for storing map pin locations in Project Compass?

Back-End Programmer

Back-End Programmers handle the creation of databases that can be used by the Front-End. In the video example it was creating a database for photos.

2. Which programmer would make Project Compass responsive on different screen sizes on the web?

Front-End Programmer

Front-End Programmers handle how the experience for the user. Each user has a different screen size and it's the Front-End's job to display information appropiately.

3. Which type of mobile programmer (Android or iOS) would be using Swift to turn Project Compass into a mobile app?

iOS Mobile Programmer

Apple developed swift as a language for it's devices. If swift becomes mainstream it may push out development for Android.

4. Which programmer would add the image to Project Compass?

Front-End Programmer

Front-End Programmers handle the user experience and place objects on webpages as fashioned by the web designer.

5. Who would evaluate information about Project Compass users to give insights for product decisions?

Data Analyst

Data Analysts make sense of data to help business officials make informed decisions.

6. Who would make requests to the Google Maps API in Project Compass?

Back-End Programmer

Back-End Programmers handle calls to the database.

7. Which programmer would need to consider hardware components to make Project Compass usable with a microphone or camera?

Back-End Programmer and Mobile Programmer

Although microphones and cameras are not mobile-only, nearly all mobile users have easy access to a microphone and camera. Back-End Programmers would handle code in an application for desktops and laptops, whereas Mobile Programmers would handle code for phones.

8. Which programmer would make a post request endpoint to add comments onto Project compass?

Back-End Programmer

Although this is similar to the user experience that Front-End often handles, post requests would need to be stored into a database that a Back-End Programmer would manage.

9. Which programmer would look at user clicking rates on Project Compass to make recommendations for external tourism sites?

Data Analyst

Generating graphs on what users click on most and compiling the data would all under the work-load of the Data Analyst
